## Get Started
### Prerequisites
- Python 3.6+
- Nvidia GPU + CUDA, CuDNN
- Tensorflow

### Installation
First use the following commands to prepare the conda environment:

```bash
conda create -n LGITM python=3.6
source activate LGITM
pip install -r requirements.txt
```
